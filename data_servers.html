GNSS Receiver Data Download Links

  • International GNSS Service (IGS)

    The IGS provides high-quality GNSS data and products that are freely available to the public.

    • IGS Data and Products
    • IGS Data Centers

  • NASA's CDDIS (Crustal Dynamics Data Information System)

    CDDIS provides access to a variety of GNSS data, including real-time data streams, hourly, daily, and high-rate data files.

    • GNSS Data
    • Data Access

  • Scripps Orbit and Permanent Array Center (SOPAC)

    SOPAC archives and distributes a wide range of GNSS data, particularly from permanent GPS stations around the world.

    • SOPAC Data Access

  • National Geodetic Survey (NGS) - OPUS

    OPUS (Online Positioning User Service) provides access to GNSS data from the NGS CORS (Continuously Operating Reference Stations) network.

    • OPUS Data

  • UNAVCO

    UNAVCO provides access to GNSS data and products, supporting geoscience research and education.

    • UNAVCO Data Access

  • Geoscience Australia

    Geoscience Australia provides GNSS data through the GNSS Data Centre.

    • GNSS Data Centre

  • European Space Agency (ESA)

    ESA provides access to GNSS data through various projects and services, such as the European GNSS Service Centre.

    • ESA GNSS Data
